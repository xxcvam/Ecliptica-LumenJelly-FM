# 音阶参考指南 (Scales Reference)

音序器现在支持 **30+ 种音阶**，涵盖西方传统、世界民族音乐和现代实验音阶。

---

## 📚 音阶分类

### 🎼 西方传统音阶 (Western Classical)

#### **Major 大调** (明亮)
- **音程结构**: 全-全-半-全-全-全-半
- **音符**: C D E F G A B C
- **情感**: 明亮、快乐、积极
- **应用**: 流行音乐、古典音乐、儿歌

#### **Minor 小调** (忧郁)
- **音程结构**: 全-半-全-全-半-全-全
- **音符**: C D Eb F G Ab Bb C
- **情感**: 忧郁、沉思、感伤
- **应用**: 流行抒情、古典音乐、电影配乐

#### **Harmonic Minor 和声小调** (戏剧性)
- **音程结构**: 全-半-全-全-半-增二度-半
- **音符**: C D Eb F G Ab B C
- **情感**: 戏剧化、异域风情、神秘
- **应用**: 古典音乐、新古典金属、中东风格

#### **Melodic Minor 旋律小调** (爵士)
- **音程结构**: 全-半-全-全-全-全-半
- **音符**: C D Eb F G A B C
- **情感**: 流畅、爵士感、现代
- **应用**: 爵士乐、融合音乐、现代古典

---

### 🎵 调式音阶 (Church Modes)

#### **Dorian 多利亚调式** (爵士/民谣)
- **音符**: C D Eb F G A Bb C
- **特点**: 比小调更明亮，有爵士感
- **应用**: 爵士乐、民谣、放克音乐
- **代表作**: 《So What》- Miles Davis

#### **Phrygian 弗里吉亚调式** (西班牙/暗黑)
- **音符**: C Db Eb F G Ab Bb C
- **特点**: 暗黑、异域、西班牙风情
- **应用**: 弗拉门戈、金属音乐、中东音乐

#### **Lydian 利底亚调式** (梦幻/飘渺)
- **音符**: C D E F# G A B C
- **特点**: 梦幻、飘渺、#4度特征音
- **应用**: 电影配乐、氛围音乐、梦幻流行

#### **Mixolydian 混合利底亚调式** (摇滚/蓝调)
- **音符**: C D E F G A Bb C
- **特点**: 蓝调感、摇滚感、降7度
- **应用**: 摇滚乐、蓝调、乡村音乐
- **代表作**: 《Sweet Child O' Mine》前奏

---

### 🎎 亚洲音阶 (Asian Scales)

#### **Japanese 日本音阶** (都节调式)
- **音符**: C Db F G Ab
- **音程**: 半-增三度-全-半
- **特点**: 5音音阶，传统日本风格
- **应用**: 游戏配乐、动画音乐、传统日本音乐

#### **Hirajoshi 日本平调子** (传统)
- **音符**: C D Eb G Ab
- **特点**: 平静、禅意、传统
- **应用**: 冥想音乐、环境音乐

#### **In Sen 日本阴旋** (神秘)
- **音符**: C Db F G Bb
- **特点**: 神秘、暗色调、现代
- **应用**: 电影配乐、现代日本音乐

#### **Chinese 中国五声音阶** (宫商角徵羽)
- **音符**: C D E G A
- **音程**: 全-全-小三度-全
- **特点**: 纯正的中国传统五声
- **应用**: 中国传统音乐、国风歌曲

---

### 🌍 中东/欧洲民族音阶 (Middle Eastern / European Folk)

#### **Arabic 阿拉伯音阶** (Hijaz)
- **音符**: C Db E F G Ab B C
- **特点**: 增二度音程，中东特色
- **应用**: 阿拉伯音乐、肚皮舞音乐、中东配乐

#### **Gypsy 吉普赛音阶** (匈牙利)
- **音符**: C D Eb F# G Ab B C
- **特点**: 两个增二度音程，充满激情
- **应用**: 吉普赛爵士、东欧民谣、Django Reinhardt风格

#### **Spanish 西班牙音阶** (Phrygian Dominant)
- **音符**: C Db E F G Ab Bb C
- **特点**: 弗拉门戈特征，降2升3
- **应用**: 弗拉门戈、西班牙吉他、拉丁音乐

#### **Jewish 犹太音阶** (Ahava Rabboh)
- **音符**: C Db E F G Ab Bb C
- **特点**: 与西班牙音阶相同，犹太传统音乐
- **应用**: 犹太礼拜音乐、克莱兹梅尔音乐

#### **Persian 波斯音阶** (神秘)
- **音符**: C Db E F Gb Ab B C
- **特点**: 异域、神秘、中东风格
- **应用**: 波斯传统音乐、中东电影配乐

---

### 🎸 布鲁斯与五声音阶 (Blues & Pentatonic)

#### **Pentatonic 五声音阶** (通用)
- **音符**: C D E G A
- **特点**: 最常用，无半音冲突
- **应用**: 摇滚即兴、流行音乐、中国音乐

#### **Pentatonic Minor 小调五声** (布鲁斯)
- **音符**: C Eb F G Bb
- **特点**: 蓝调感、摇滚感
- **应用**: 布鲁斯、摇滚独奏、即兴演奏

#### **Blues 布鲁斯音阶** (经典)
- **音符**: C Eb F F# G Bb
- **特点**: 包含"蓝调音"(#4/b5)
- **应用**: 布鲁斯、爵士、摇滚
- **代表**: B.B. King, Eric Clapton 风格

---

### 🌈 特殊/对称音阶 (Special / Symmetrical Scales)

#### **Whole Tone 全音阶** (印象派)
- **音符**: C D E F# G# Bb
- **音程**: 全-全-全-全-全-全
- **特点**: 没有主音感，梦幻、漂浮
- **应用**: 印象派音乐、电影配乐
- **代表**: Debussy《沉没的教堂》

#### **Chromatic 半音阶** (全部音)
- **音符**: 所有12个半音
- **特点**: 完全自由，无调性
- **应用**: 现代古典、实验音乐、过渡

#### **Diminished 减音阶** (对称)
- **音符**: C D Eb F Gb Ab A B C
- **音程**: 全-半-全-半-全-半-全-半
- **特点**: 对称结构，爵士常用
- **应用**: 爵士即兴、减和弦上

#### **Augmented 增音阶** (六声)
- **音符**: C Eb E G Ab B
- **特点**: 增三和弦叠加，对称
- **应用**: 爵士、现代音乐

#### **Octatonic 八声音阶** (爵士)
- **音符**: C Db Eb E F# G A Bb
- **音程**: 半-全-半-全-半-全-半-全
- **特点**: 对称，与减音阶相反顺序
- **应用**: 现代爵士、古典现代派

---

### 🎺 爵士音阶 (Jazz Scales)

#### **Bebop 比波普音阶** (爵士)
- **音符**: C D E F G A Bb B C (8音)
- **特点**: 在大调基础上加经过音
- **应用**: Bebop爵士、快速即兴
- **代表**: Charlie Parker, Dizzy Gillespie

---

### 🔮 神秘/实验音阶 (Mystic / Experimental)

#### **Prometheus 普罗米修斯音阶** (神秘)
- **音符**: C D E F# A Bb
- **特点**: Scriabin创造，神秘、现代
- **应用**: 现代古典、实验音乐、氛围

#### **Tritone 三全音音阶** (紧张)
- **音符**: C Db E F# G Bb
- **特点**: 两个三全音，极度紧张
- **应用**: 实验音乐、恐怖配乐

#### **Enigmatic 谜语音阶** (奇特)
- **音符**: C Db E F# Ab Bb B C
- **特点**: Verdi使用，神秘、不稳定
- **应用**: 现代配乐、实验音乐

## 🎹 使用建议

### 按情感选择

- **快乐/明亮**: Major, Lydian, Pentatonic
- **忧郁/沉思**: Minor, Phrygian, Blues
- **神秘/异域**: Arabic, Gypsy, Persian, Prometheus
- **梦幻/飘渺**: Lydian, Whole Tone, Pentatonic
- **爵士/现代**: Dorian, Mixolydian, Bebop, Diminished
- **传统民族**: Japanese, Chinese, Spanish, Jewish

### 按流派选择

- **流行音乐**: Major, Minor, Pentatonic
- **摇滚**: Minor Pentatonic, Blues, Mixolydian
- **爵士**: Dorian, Bebop, Diminished, Melodic Minor
- **电影配乐**: Harmonic Minor, Lydian, Whole Tone
- **电子/氛围**: Whole Tone, Chromatic, Prometheus
- **世界音乐**: Arabic, Japanese, Gypsy, Chinese

### 按难度选择

- **初学者**: Major, Minor, Pentatonic
- **中级**: Dorian, Mixolydian, Blues, Japanese
- **高级**: Diminished, Bebop, Octatonic, Enigmatic
- **实验**: Whole Tone, Chromatic, Tritone, Prometheus

## 💡 创作技巧

1. **对比使用**: 在同一曲目中切换音阶（如从Major到Minor）
2. **融合民族**: 使用Arabic或Japanese音阶加入异域色彩
3. **实验探索**: 尝试Whole Tone或Diminished创造独特音色
4. **爵士进阶**: 在Dorian和Bebop之间切换增加即兴感
5. **氛围营造**: Lydian和Prometheus适合制造飘渺氛围

## 📊 技术说明

### 音程表示法
- 数字表示相对根音的半音数
- `[0, 2, 4, 5, 7, 9, 11]` = C大调音阶
- 0=根音, 2=大二度, 4=大三度, 5=纯四度, 7=纯五度, 9=大六度, 11=大七度

### 量化机制
音序器使用**最近音符量化算法**：
- 将输入音高转换为MIDI音符
- 在选定音阶内找到最接近的音符
- 跨八度自动处理
